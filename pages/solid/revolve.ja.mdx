# Revolve（リボルブ）

<Reference shortcut="" command="Revolve" button="コマンドバー右" buttonIcon="button-revolve"/>

リージョンまたはカーブをリボルブできます。ソリッドまたはシートが作成されます。

## 基本的な使い方

1. リボルブするリージョンまたはカーブを選択します。
2. コマンドバー右のボタン <ButtonIcon type="button-revolve" /> をクリックします。
3. 回転の軸を指定します。
   - 最初に始点をクリックし、次に終点をクリックした2点をつなぐ線が回転の軸になります。
      - <Kst>キーボードショートカット</Kst>
         - <kbd>Shift + T</kbd> **Thickness:**
         - <kbd>A</kbd> **Angle:** マウスを動かして回転角度を指定します。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
