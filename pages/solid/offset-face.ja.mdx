# Offset Face（フェイスをオフセットする）

<Reference shortcut="Shift + O" command="Offset Face" button="" />

フェイスをオフセットすることができます。

## 基本的な使い方

1. オフセットしたいフェイスを選択します。
2. フェイスが選択されると、デフォルトのコマンドであるOffset Faceが自動的に実行されます。
3. 距離を指定するために黄色い点を動かします。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Distance:** 距離を指定します。
        - **Degrees:** 角度を指定します。
        - **Grow:**
            - 凹んだ内側のフェイスをオフセットするとき:
            - **Moving:** オフセットが外壁に到達したら、壁全体を移動します。
            - **Fixed:** 外壁で停止します。
            - **None:** 外壁の影響を受けず、オフセットが進み続けます。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>D</kbd> **Distance** カーソルを動かして距離を指定します。
        - <kbd>A</kbd> **Angle adjacent** カーソルを動かして角度を指定します。
        - <kbd>Tab</kbd> **Grow mode** Grow modeを切り替えます。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。

## 依存オフセット

選択された面を他の面に合わせ、オフセットを1回の操作で行う。

1. オフセットしたい面を選択してください。
2. 面が選択されると、デフォルトコマンドである **Offset Face** が自動的に実行されます。
3. 面をオフセットし始め、その後オフセットギズモを解除してください。
4. マッチさせたい面を選択してください。必要に応じて、オフセットギズモをクリックしてドラッグするか、オフセット値を入力して面をオフセットしてください。
5. コマンドダイアログで OK をクリックするか、右クリックで確認してください。
