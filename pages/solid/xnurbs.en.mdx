# xNURBS

<Reference shortcut="" command="xNURBS" buttonIcon="-"/>

Create smooth and complex surfaces.

## Basic Usage

1. Select two or more edges you want to frame the surface.
2. Use the **xNURBS** command in the Command Palette.
3. Set the options in the Command Dialog.
    - <Kst>Command Dialog</Kst>
          - **Continuity:** Specify the tolerance of surface CV count. Lower value results in large number of surface CVs and higher value results in small number of CVs. This determines the surface output quality.
          - **Quad sided:** Specify the U and V extend value.
          - **Flatness:** Shrink projecting rows and columns of CVs to fit more closely to the boundary edges of the trimmed surface.
