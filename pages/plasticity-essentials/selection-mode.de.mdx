# Auswahlmodus

Wechseln Sie den Auswahlmodus je nach dem Objekt, das Sie auswählen möchten.
Um den Auswahlmodus zu wechseln, verwenden Sie die Tasten in der oberen linken Ecke des Bildschirms oder Tastenkombinationen.

![Auswahlmodus Schaltflächen](/img/selection-mode-buttons.png)

| Befehlsname                         | Tastenkombination       | Schaltfläche                   | Hinweis                                                    |
| ----------------------------------- | ----------------------- | ------------------------------ | ---------------------------------------------------------- |
| Selection mode: set Control Point   | <kbd>1</kbd>            | (1)                            | Ermöglicht die Auswahl von Control-Point.                  |
| Selection mode: set edge            | <kbd>2</kbd>            | (2)                            | Ermöglicht die Auswahl von Edge oder Curve.                |
| Selection mode: set face            | <kbd>3</kbd>            | (3)                            | Ermöglicht die Auswahl von Face oder Region.               |
| Selection mode: set solid           | <kbd>4</kbd>            | (4)                            | Ermöglicht die Auswahl von Solid oder Sheet.               |
| Selection mode: set all             | <kbd>Tab</kbd>          | (5)                            | Ermöglicht die Auswahl aller Objekttypen.                  |
| Selection: toggle Control Point     | <kbd>Umschalt + 1</kbd> | <kbd>Umschalt</kbd> + Klick (1)| Wechselt den Control Point Auswahlmodus.                   |
| Selection: toggle edge              | <kbd>Umschalt + 2</kbd> | <kbd>Umschalt</kbd> + Klick (2)| Wechselt den Edge Auswahlmodus.                            |
| Selection: toggle face              | <kbd>Umschalt + 3</kbd> | <kbd>Umschalt</kbd> + Klick (3)| Wechselt den Face Auswahlmodus.                            |
| Selection: toggle solid             | <kbd>Umschalt + 4</kbd> | <kbd>Umschalt</kbd> + Klick (4)| Wechselt den Solid Auswahlmodus.                           |
